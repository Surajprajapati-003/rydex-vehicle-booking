# 🚖 Rydex – Vehicle Booking & Partner Management Platform

Rydex is a full-stack vehicle booking and partner onboarding platform that enables users to book vehicles, register as partners, upload verification documents, and allows administrators to manage approvals through a centralized dashboard.

---

## 🔗 Live Demo

👉 https://3-rydex-xi.vercel.app/

---

## ⭐ Features

### 👤 User Module

* Secure Authentication
* Profile Management
* Vehicle Booking

### 🤝 Partner Module

* Partner Registration
* Multi-Step Onboarding
* Aadhaar Upload
* RC Upload
* Driving License Upload
* Partner Verification Tracking

### 🚗 Vehicle Module

* Vehicle Registration
* Vehicle Management
* Vehicle Availability Tracking

### 🛠 Admin Module

* Admin Dashboard
* Partner Review System
* Document Verification
* Approval & Rejection Workflow

## 🚀 Installation Guide

### Clone Repository

```bash
git clone https://github.com/Surajprajapati-003/rydex-vehicle-booking.git
```

Move into project:

```bash
cd rydex-vehicle-booking
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```

---

## 🔐 Environment Variables

Create `.env.local`

```env
MONGODB_URL=your_mongodb_connection_string

AUTH_SECRET=your_auth_secret

AUTH_GOOGLE_ID=your_google_client_id

AUTH_GOOGLE_SECRET=your_google_client_secret

EMAIL=your_email

PASS=your_email_app_password

CLOUDINARY_CLOUD_NAME=your_cloudinary_name

CLOUDINARY_API_KEY=your_cloudinary_api_key

CLOUDINARY_API_SECRET=your_cloudinary_secret

NEXT_PUBLIC_ZEGO_APP_ID=your_zego_app_id

NEXT_PUBLIC_ZEGO_SERVER_SECRET=your_zego_secret

NEXT_PUBLIC_SOCKET_SERVER_URL=your_socket_server_url

NEXT_PUBLIC_GEOAPIFY_API_KEY=your_geoapify_key

RAZORPAY_KEY_ID=your_razorpay_key

RAZORPAY_KEY_SECRET=your_razorpay_secret

NEXT_PUBLIC_RAZORPAY_KEY_ID=your_public_razorpay_key

GEMINI_API_URL=your_gemini_api_url
```
